# Prism - Native AI Chat for macOS

![Prism App Icon](AppIcon.png)

## Overview

Prism is a cutting-edge, native macOS application designed to be your ultimate AI companion. Built with SwiftUI, it offers a seamless and visually stunning "Liquid Glass" interface that feels right at home on your Mac. 

Unlike other chat apps, Prism isn't locked to a single AI provider. It empowers you to choose the right tool for the job:
*   **Google Gemini:** Harness the power of Google's latest cloud models for complex reasoning and multimodal analysis.
*   **Ollama:** Run privacy-focused local models (like Llama 3, DeepSeek, and Mistral) directly on your machine with zero data leaving your device.
*   **Apple Shortcuts:** Deeply integrate with your Mac's system automation to control apps and workflows.

Whether you're solving complex math equations with beautiful LaTeX rendering, analyzing images, or just need a quick answer via the lightweight "Quick Chat" window, Prism delivers with streaming performance and a user-centric design.

## Key Features

*   **Multi-Model Intelligence:** Switch instantly between Gemini, Local Ollama models, and System Shortcuts.
*   **Streaming Responses:** Experience fast, character-by-character output just like the web interfaces.
*   **Thinking Process:** For reasoning models (like DeepSeek R1), view the internal "thought process" in a collapsible section.
*   **Rich Media Support:** 
    *   **LaTeX Math:** Renders inline (`$x^2$`) and display (`$$. . .$$`) equations perfectly.
    *   **Markdown:** Full support for code blocks, tables, and formatting.
    *   **Images:** Drag & drop image analysis.
*   **Native UI:** Glassmorphic design, swipe gestures, and smooth animations.

## Getting Started

Once installed, click the **Gear Icon** (Settings) to set up your AI providers:

1.  **Google Gemini:** Paste your API key from [Google AI Studio](https://aistudio.google.com/).
2.  **Ollama:** Ensure [Ollama](https://ollama.com/) is running (`ollama serve`) and enter your model name (e.g., `llama3`).
3.  **Shortcuts:** Enter the exact name of any Apple Shortcut you wish to trigger.
